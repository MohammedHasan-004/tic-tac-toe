Tic Tac Toe (X-O Game) – ReactJS

This is a simple Tic Tac Toe (X-O) game built using ReactJS.
The project allows two players to play against each other with customizable player names.

🚀 Features:
Player vs Player mode (2 players)
Ability to change player names
Interactive game board built with reusable React components
Winner detection with a popup message
Draw detection (no winner case)
Clean and simple UI
🧠 What I Learned:
Building reusable components in React
Managing and rendering list items using React
Handling state and user interactions
Structuring a small React project
